# Load Test Reports

## Objectif
Regrouper les rapports issus des tests de charge.

## Format recommandé
- Résumé des résultats
- Points de rupture identifiés
- Recommandations d'optimisation
- Plan de scaling

## Emplacement
Les rapports doivent être ajoutés dans ce dossier sous forme de `*.md` ou `*.pdf`.

# Load Test Scripts

## Objectif
Grouper les scripts d'exécution des tests de charge.

## Contenu possible
- `run-load-test.sh`
- `generate-scenario.sh`
- `collect-results.sh`

## Utilisation
1. Ajouter des scripts de test automatisés.
2. Lancer les scripts depuis le dossier `load-tests/`.
3. Stocker les résultats dans `results/`.

# Résultats des tests de charge

## Objectif
Stocker les résultats bruts et synthétiques des tests de charge.

## Contenu recommandé
- `results-YYYY-MM-DD.md`
- `scenario-name-results.md`
- `summary-dashboard.md`

## Utilisation
1. Documenter les métriques observées.
2. Lister les points de saturation.
3. Archiver les logs et captures d'écran de benchmark.

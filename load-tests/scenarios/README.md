# Load Test Scenarios

## Objectif
Décrire les scénarios de charge à exécuter pour JoinHashcode.

## Scénarios
- connexion
- onboarding
- quiz
- dashboard
- missions
- squads
- notifications

## Instructions
1. Créer un script de test par scénario.
2. Exécuter les tests à plusieurs paliers d'utilisateurs.
3. Enregistrer les résultats dans `../results/`.

# Optimisations Database

## Objectif
Lister les optimisations liées à Prisma et PostgreSQL.

## Contenu recommandé
- Indexation
- Requêtes optimisées
- Transactions courtes
- Partitionnement / agrégations pré-calculées
- Pooling et connexion

## Utilisation
1. Documenter chaque optimisation.
2. Indiquer les index SQL recommandés.
3. Lier aux audits PostgreSQL.

# Optimisations Frontend

## Objectif
Lister les optimisations spécifiques au frontend.

## Contenu recommandé
- Bundle splitting
- Dynamic imports
- Memoization
- Caching
- Réduction des re-renders
- Optimisation des composants Radix/UI

## Utilisation
1. Documenter chaque optimisation.
2. Évaluer le gain et la difficulté.
3. Lier aux audits et aux résultats de benchmark.

# Metrics

## Objectif
Lister les métriques suivies pour JoinHashcode.

## Contenu recommandé
- `application.response_time`
- `application.error_rate`
- `nodejs.heap_usage`
- `nodejs.event_loop_lag`
- `postgres.active_connections`
- `docker.cpu_usage`
- `vps.disk_usage`

## Utilisation
- Définir les métriques dans le système de monitoring.
- Standardiser les noms de métriques.
- Documenter les unités et les seuils.

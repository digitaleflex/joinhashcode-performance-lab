# Alerts

## Objectif
Définir des règles d'alerte pour détecter les anomalies rapidement.

## Contenu
- Alerte latence
- Alerte erreurs
- Alerte utilisation CPU/RAM
- Alerte base de données
- Alerte redémarrage Docker

## Recommandations
- Définir des niveaux Warning et Critique.
- Documenter les actions à prendre pour chaque alerte.

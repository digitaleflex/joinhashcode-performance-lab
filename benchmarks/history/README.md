# Historique des benchmarks

## Objectif
Suivre l'évolution des performances dans le temps.

## Contenu recommandé
- `baseline-YYYY-MM-DD.md`
- `loadtest-YYYY-MM-DD.md`
- `frontend-metrics-YYYY-MM-DD.md`

## Utilisation
1. Ajouter un enregistrement à chaque mesure de référence.
2. Comparer les résultats aux baselines précédentes.
3. Noter les changements d'environnement.

# Rapports d'audit

## Objectif
Centraliser les audits terminés et rendre leur accès facile.

## Contenu recommandé
- `audit-YYYY-MM-DD-application.md`
- `audit-YYYY-MM-DD-database.md`
- `audit-YYYY-MM-DD-infrastructure.md`

## Utilisation
1. Copier un template dans un nouveau fichier.
2. Remplir le rapport avec les conclusions et recommandations.
3. Versionner le rapport.
